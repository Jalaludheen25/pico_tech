# Website Color Theme Analysis: Candela Medical

I have analyzed the color theme of [candelamedical.com](https://candelamedical.com/). The website uses a sophisticated, premium "Medical Aesthetic" theme defined by deep purples, vibrant magenta accents, and dark neutral tones.

## Core Color Palette

| Color Role | Hex Code | Visual | Usage |
| :--- | :--- | :--- | :--- |
| **Primary Brand Purple** | `#9A1F8F` | <div style="width: 20px; height: 20px; background-color: #9A1F8F; border: 1px solid #000;"></div> | Core branding, icons, and active states. |

| **Snow White** | `#F9F9F9` | <div style="width: 20px; height: 20px; background-color: #F9F9F9; border: 1px solid #000;"></div> | Background for content-heavy sections. |
